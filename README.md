# AI Tools Workshop

A starter repository and environment configuration set up for the AI Tools Workshop.

## Overview

This repository acts as the base working directory for participants. It includes the necessary `.gitignore` templates and environment scaffolding (via `venv_aiworkshop`) to ensure a smooth, standardized setup before diving into the practical AI experiments.

## Setup Instructions

1. Clone the repository.
2. Activate the virtual environment:
   ```bash
   # On Windows
   venv_aiworkshop\Scripts\activate
   # On macOS/Linux
   source venv_aiworkshop/bin/activate
   ```
3. Install any workshop-specific dependencies as instructed during the session.
